# 单行远程控制工具栏改版

## 问题

连接后的悬浮工具栏原本以图标按钮为主，但鼠标模式、图像清晰度和颜色位数使用了三个常驻下拉框。它们破坏了视觉一致性，并使 `WrapPanel` 在空间不足时换成两行，遮挡远程画面。

## 方案

- 三个下拉框改为与现有控件一致的 40x38 图标按钮。
- 点击图标后打开单层菜单；当前选项使用勾选标记，按钮 Tooltip 同时显示当前值。
- 工具栏改用不可换行的水平布局，在窗口最小宽度和 150% DPI 下仍保持一行。
- 协议命令、失败回滚、刀片切换、重连和权限禁用逻辑保持不变。
- 固件不支持的颜色位数在菜单中禁用，不展示为可执行操作。

## 验收

- 连接后的悬浮工具栏只有图标控件并保持单行。
- 鼠标模式、清晰度和色深的当前状态可以从菜单勾选与 Tooltip 看出。
- 三类设置仍发送原有协议命令，失败时恢复实际会话状态。
- 普通用户与只读会话无法使用无权限的设置。
- 桌面冒烟测试只连接 `127.0.0.1` 回环服务，不调用任何电源、挂载、弹出或 USB 重置动作。
